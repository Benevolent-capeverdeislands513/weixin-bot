## 📦 What this app does

weixin-bot lets you run a WeChat AI bot on your Windows PC with one simple download. It can connect to common AI services like OpenAI, Claude, Gemini, and Ollama. You do not need to install extra tools first.

Use it to:

- Reply to WeChat messages with AI
- Switch between different AI providers
- Run a local model with Ollama
- Keep setup simple for home or small team use
- Start fast with a clean Windows app flow

## 💻 Before you start

You only need:

- A Windows PC
- An internet connection for the first download
- A WeChat account
- A release file from the download page
- An API key if you want to use OpenAI, Claude, or Gemini

If you plan to use Ollama, make sure Ollama is running on your PC before you start the bot.

- Your AI provider

Common provider choices:

- OpenAI for general chat tasks
- Claude for long text and careful replies

If the app shows a settings screen, fill in the fields one by one and save your changes before starting the bot

## 🧩 How to use it

Once the bot is running:

1. Send a message to the WeChat account linked to the bot
2. The bot reads the message
3. The selected AI model creates a reply
4. The reply goes back into the chat

You can use it for:

- Quick answers
- Simple task help
- Message drafting
- Basic support replies
- Personal chat automation

## 🔐 API key setup

2. Copy your API key
3. Paste it into the app settings

Keep your API key private. Do not share it in chat or post it in public places

## 🛠️ Troubleshooting

If the app does not start:

- Check that you downloaded the right Windows file
- Extract the ZIP if the release came in a compressed file
- Run the app from the extracted folder
- Try again after a fresh download

If the bot does not reply in WeChat:

- Check that the bot is running
- Make sure your API key is correct
- Confirm that your model or provider is selected
- Check your internet connection if you use a cloud provider

If Ollama does not work:

- Make sure Ollama is open
- Check that the model is installed
- Confirm the model name in the app
- Restart both Ollama and weixin-bot

If Windows blocks the file:

- Right-click the file
- Open its properties
- Look for an unblock option if Windows shows one
- Run the app again
